# AgentFrontEnd - AI 聊天应用

一个基于 React + TypeScript + Material-UI 的现代化 AI 聊天应用前端。

## 🚀 项目状态

✅ **已完成** - 项目基础架构和核心功能已实现，开发服务器运行在 http://localhost:3000

## 🛠 技术栈

- **前端框架**: React 18.2.0 + TypeScript 5.2.2
- **构建工具**: Vite 4.5.0
- **UI 组件库**: Material-UI 5.14.20
- **样式方案**: Tailwind CSS 3.3.5 + MUI 主题系统
- **状态管理**: Zustand 4.4.7
- **数据获取**: TanStack Query 5.8.4
- **HTTP 客户端**: Axios 1.6.2

## ✨ 主要功能

### 🎯 已实现功能

1. **完整的聊天界面**
   - 现代化的Material Design风格
   - 响应式布局，支持桌面和移动端
   - 实时消息流式传输（SSE）
   - 消息气泡带头像和时间戳
   - 消息操作（复制、更多选项）

2. **会话管理**
   - 创建新对话
   - 会话列表展示
   - 删除单个或批量会话
   - 会话时间格式化显示
   - 最后消息预览

3. **用户界面**
   - 应用头部导航
   - 聊天页面和设置页面
   - 错误边界处理
   - 加载状态指示
   - 成功/错误提示

4. **设置管理**
   - 界面设置（深色模式、自动保存、通知）
   - API配置（URL、Token数、温度参数）
   - 数据管理（保存、重置、清除）
   - 应用信息展示

## 🚀 快速开始

### 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
# 浏览器打开 http://localhost:3000
```

### 构建生产版本

```bash
npm run build
```

## ⚙️ 配置

### 环境变量

创建 `.env` 文件：

```env
# API 基础URL
VITE_API_BASE_URL=http://localhost:8080

# 应用配置
VITE_APP_TITLE=Agent Frontend
VITE_APP_VERSION=0.0.1
```

## ✨ 新功能特性

### 🎯 智能布局切换
- **初始状态**: 聊天输入框居中显示，提供简洁的开始界面
- **对话状态**: 用户开始对话后，输入框自动移动到底部，显示完整的聊天界面
- **状态记忆**: 智能记住用户的对话状态，提供连贯的用户体验

### 🎨 简约设计美学
- **去除冗余**: 移除了"AI智能助手"文字和机器人图标
- **专注内容**: 界面更加简洁，突出对话内容
- **Material Design 3**: 遵循最新的 Material Design 设计规范

### 📱 可折叠侧边栏
- **左侧会话列表**: 可以通过按钮折叠/展开
- **平滑动画**: 使用 Material-UI Collapse 组件实现流畅的折叠效果
- **响应式设计**: 适配不同屏幕尺寸

## 🎨 设计原则

### Material Design 3 核心理念
1. **材质隐喻**: 界面元素具有物理世界的质感
2. **大胆图形设计**: 清晰的视觉层次和焦点
3. **有意义的动画**: 提供空间连续性和功能反馈

### 简约设计哲学
- **内容优先**: 突出对话内容，减少视觉干扰
- **功能导向**: 每个元素都有明确的功能目的
- **一致性**: 统一的设计语言和交互模式
